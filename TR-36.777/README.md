# TR 36.777

**Study on Enhanced LTE Support for Aerial Vehicles**

Release 15, current version tracked in this repo: v15.0.0.

Source: https://www.3gpp.org/dynareport/36777.htm

Coverage of this TR in the library is currently **Annex B (Channel modelling details)** only — the aerial-UE channel model, expressed largely as modifications and extensions to TR 38.901's UMa/UMi/RMa models for above-rooftop UE heights. See [INDEX.md](../INDEX.md) at the repo root for current section-by-section status.

> **Note on the accompanying spreadsheet.** The source document ships with `36777-f00_3.xlsx` (large-scale calibration CDFs for UMi-AV/UMa-AV/RMa-AV — coupling loss and geometry). That is Annex C *calibration output*, not Annex B channel-model parameters, and is treated as reference-only (kept in `references/`, not structured into this library) — the same way `references/3gpp-R1-2509126/` is handled.
